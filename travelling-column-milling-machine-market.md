<p><p>Travelling column milling machines are commonly classified into two main types: horizontal type and floor type. The horizontal type refers to machines where the spindle is parallel to the worktable, allowing the cutting tool to move horizontally. These machines are often used for creating horizontal grooves or slots on a workpiece. On the other hand, the floor type refers to machines where the worktable moves vertically, allowing for vertical milling operations. This type is suitable for cutting deep grooves or holes into large workpieces.</p></p>
